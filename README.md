# StarkSwirl-UI
This is built using
- NextJS
- TypeScript
- starknet-js
- starknet-react
- TailwindCSS
- ShadCn
- Aceternity UI
  
## Getting Started

First install the dependencies:

```bash
pnpm i
#or
yarn
#or
npm i
```
Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then, open `localhost:3000`
